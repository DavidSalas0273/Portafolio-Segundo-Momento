Carpeta para todas las imágenes usadas en el proyecto.

Convenciones recomendadas:
- Nombres en minúsculas, sin espacios: `avatar.jpg`, `project-1.png`, `hero-portrait.jpg`.
- Formatos sugeridos: WebP para producción (`.webp`) por su eficiencia; JPG/PNG para edición y compatibilidad.
- Tamaños: guarda originales en `assets/images/originals/` y versiones optimizadas en `assets/images/optimized/`.
- Si usas una imagen en el hero, ponla como `assets/images/avatar.jpg` o actualiza la ruta en `index.html`.

Ejemplo de estructura:
- assets/images/originals/avatar-2048.jpg
- assets/images/optimized/avatar.jpg (800x800, comprimida)

Puedes añadir aquí tus imágenes. Si quieres, puedo crear subcarpetas (`originals`, `optimized`) y un script de optimización más adelante.
